## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Publications Persistence

- Frontend-only: sem API, sem backend e sem base de dados externa.
- Fonte de verdade: [src/data/publications.json](src/data/publications.json)
- A secao publica e o painel admin leem os dados deste ficheiro JSON.
- Para adicionar/editar/remover publicacoes, edite manualmente o JSON.
